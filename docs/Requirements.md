# 4. 요구사항 명세

## 4.1 필수 과제

### **4.1.1 회원가입 / 로그인**

- username과 password로 회원가입 및 로그인을 할 수 있습니다.
- 로그인 성공 시 발급된 토큰을 저장하여 이후 요청에 활용합니다.
- 입력값이 모두 채워져야 버튼이 활성화됩니다.
- 로그인 실패 시 경고 문구를 표시합니다.

### **4.1.2 블로그 목록 페이지(홈)**

- 전체 포스트 목록을 표시합니다. 각 포스트는 썸네일, 제목, 작성자, 작성일을 표시합니다.
- 포스트가 없을 경우 "아직 작성된 글이 없습니다." 문구를 표시합니다.

### **4.1.3 포스트 상세 페이지**

- 제목, 작성자, 작성일, 썸네일(있을 경우), 본문 내용을 표시합니다.
    - 썸네일이 없을 경우 랜덤한 이미지를 노출합니다.
- 내가 작성한 포스트일 경우 수정, 삭제 버튼이 표시됩니다.
- 삭제 시 확인 모달을 표시하며, 확인 후 목록 페이지로 이동합니다.

### **4.1.4 포스트 작성 / 수정 페이지**

- 제목과 본문이 모두 입력되면 `발행` 버튼이 활성화됩니다.
- 수정 페이지에서는 기존 내용이 입력창에 미리 채워집니다.

## 4.2 선택(도전) 과제

### **4.2.1 포스트 검색**

- 제목 또는 작성자로 포스트를 검색할 수 있습니다.
- 검색어와 일치하는 단어는 위니브 메인 컬러로 표시됩니다.
